# p5-TS-Vite-template


### About
This is a template to use a [p5.js](https://github.com/processing/p5.js) canvas in a project with Typescript and Vite. 

### Usage
Clone this repo and run 
```
npm i
```
to install dependencies. Then 
```
npm run dev
``` 
to start the dev server.
